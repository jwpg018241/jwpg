Scroll

Navigation

① Desktop - 동일한 계층으로 스크롤 되어 이동합니다.
② Mobile - 하위 요소(네비게이션)가 상위 요소 위로 확장이 됩니다.

Components Reflow

UI는 기기별로 Reflow 되도록 레이아웃을 변경됩니다.
Desktop의 수평 목록은 Mobile 화면에서 수직 목록으로 Reflow 됩니다.

Components Transitions

탐색 전환은 사용자가 홈 화면에서 컨텐츠 화면으로 이동하는 등 화면 사이를 이동할 때 발생합니다.

① Desktop - Ajax 팝업을 통해 하위 요소(Components)가 상위 요소 위로 팝업이 됩니다.
② Mobile - 하위 요소(Components)가 상위 요소 위로 확장이 됩니다.
