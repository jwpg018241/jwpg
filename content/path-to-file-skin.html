LG Skincare

데이터 중심 미학

Skincare의 UI는 데이터보기, 조사 및 이해에 중점을 둡니다. 인포 그래픽은 사용자가 세부 정보를 포함한 요약을 볼 수 있도록 합니다. UI는 정보에 중점두고 직접적으로 전달합니다.

구조

Flow 구조

Skincare는 Flow 구조를 사용합니다. Flow 구조는 화면을 순서대로 액세스할 수 있는 구조입니다.
Flow 구조는 Skincare에 가장 적합한데, 관심 있는 메뉴를 사용자가 실행 후 일련의 과정을 통해서 사용자에게 필요한 정보를 제공하기 때문입니다.


데이터의 시각화

그래프의 사용

정보의 내용이나 보여주려는 의도에 따라서 하나의 그림으로 보여주는 게 훨씬 효과적일 때가 있습니다.
데이터 시각화 그래프를 만드는 이유는 데이터의 의미를 시각적으로 빠르게 파악하기 위함입니다.

컨텐츠 구성

카드형 컨텐츠

관리 방식으로 조밀하게 구성된 대량의 콘텐츠를 카드 형식으로 구분하여, 사용자가 이해하기 쉬운 형태로 가공됩니다.
이런 측면에서 사용자는 흥미 있어 하는 콘텐츠를 쉽게 볼 수 있고, 자신이 원하는 것에 적극적으로 관여할 수 있도록 도와줍니다.

UI Editorial Design

정보의 가공

많은 정보는 사용자의 눈에 잘 들어오지 않습니다. 그 이유는 정보의 양이 많을수록 사용자는 읽지 않기 때문입니다. 그럴 때는 그것들의 콘텐츠의 우선순위를 분류하고 시각적으로 구조화하는 정보의 가공 과정이 필요합니다.
